# potion-app
Aplicativo para vender potions tibianas.

### gif-demo

### requirements

* node
  * ionic
  * cordova
  * gulp
  * bower

### installing

1. `npm install bower cordova ionic gulp -g`
  Resolves global deps
2. `npm install`
  Resolves project deps

### running

1. `gulp build watch`
2. `ionic serve --lab`
